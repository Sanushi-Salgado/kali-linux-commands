# 💻 PostgreSQL Commands 🐉

---

## 🛠️ Usage

```bash
pg_lsclusters                                          # 🛠️ Check PostgreSQL cluster status

sudo pg_ctlcluster <version> main start                # 🚀 Start the cluster (if it's down)

sudo pg_ctlcluster <version> main restart              # 🔄 Restart the cluster
```
