# 💻 Github-Endpoints Commands 🐉

---

## 🔄 Installation

```bash
go install github.com/gwen001/github-endpoints@latest      #  📦 Install Github-Endpoints using Go 
```

---

## ✅ Verification 

```bash
which github-endpoints									   # 📍 Show the path to the installed Github-Endpoints executable

github-endpoints -h										   # ❓ Display help information for Github-Endpoints
```

---

## 🛠️ Usage

```bash
export GITHUB_TOKEN=ghp_xxxxxxxxxxxxxxxxx				   # ➕ Configure Github token to use Github-Endpoints

github-endpoints -d <domain>							   # 🌐 Get endpoints related to a domain
```
