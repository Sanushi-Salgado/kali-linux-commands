# 💻 KnockKnock  Commands 🐉

---

## ✅ Verification 

```bash
which knockknock      											# 📍 Show the path to the installed KnockKnock executable
```

---

## 🛠️ Usage

```bash
knockknock														# 🚀 Launch knockknock
 
knockknock -n <domain>                     						# 🌐 Get reverse whois information related to a domain
```
