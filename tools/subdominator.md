# 💻 Subdominator Commands 🐉

---

## ✅ Verification 

```bash
which subdominator                               # 📍 Show the path to the installed Subdominator executable

subdominator -h                                  # ❓ Display help information for Subdominator 
```

---

## 🛠️ Usage

```bash
subdominator -d <domain>                          # 🌐 Get subdomains related to a domain
```
