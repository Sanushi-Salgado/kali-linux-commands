# 💻 Cencli Commands 🐉

---

## 🔄 Installation

```bash
sudo apt install cencli -y										# 📦 Install Cencli via APT package manager
```

---

## ✅ Verification 

```bash
which cencli      												# 📍 Verify installation & show the path of Cencli executable
```

---

## 🛠️ Usage

```bash
cencli                     										# 🚀 Launch Cencli 
cencli credits             										# 💳 View API usage / credits
cencli config auth         										# 🔐 View configured Personal Access Tokens (PAT)
cencli config auth add     										# ➕ Add a new Personal Access Token (PAT)
cencli view <IP address>       									# 🌐 View Censys information related to an IP address
```
