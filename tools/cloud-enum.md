# 💻 Cloud Enum Commands 🐉

---

## ✅ Verification 

```bash
which cloud_enum      											# 📍 Show the path to the installed Cloud Enum executable

cloud_enum -h													# ❓ Display help information for Cloud Enum
```

---

## 🛠️ Usage

```bash
cloud_enum -k <organization / keyword>       					# 🌐 Get cloud assets related to an organization / keyword

cloud_enum -kf <keywords file> -qs								# 🌐 Get cloud assets using a keywords list while suppressing status output for quieter execution
```
