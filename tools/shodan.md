# 💻 Shodan Commands 🐉

---

## 🛠️ Usage

```bash
curl "https://api.shodan.io/api-info?key=<API key>"                     				# 💳 View API usage / credits

curl "https://api.shodan.io/shodan/host/<IP address>?key=<API key>"             		# 🌐 Get information related to an IP address        								   							
```
