# 💻 Semgrep Commands 🐉

---

## 🔄 Installation

```bash
pipx install semgrep											# 📦 Install Semgrep using pipx (isolated environment)
```

---

## ✅ Verification 

```bash
which semgrep   												# 📍 Show the path to the installed Semgrep executable
semgrep --version												# 🛠️ Show the installed Semgrep version
```
